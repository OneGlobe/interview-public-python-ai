# DevSecOps Engineer Interview

## Interview Guidelines

- :white_check_mark: Google / Internet Search
- :white_check_mark: AI questions w/o copy-paste (eg. Claude, ChatGPT, Perplexity)
- :x: AI coding agents, AI autocomplete (eg. Copilot, Cursor, Claude Code)
- :x: Changes made prior to the interview will not be accepted

## Preparation

- [ ] Clone this repository to your local machine
- [ ] Install Azure CLI
- [ ] Install Terraform CLI
- [ ] Install Kubernetes CLI (kubectl)

## Task

Your task is to deploy our chat application (backend, frontend, and database) to Kubernetes on Azure.

- The infrastructure must be managed as code, it must reside in the repo alongside the applications.
- Migrations must be run by the deployment process.

You will be provided with Azure credentials. Be prepared to run your solution.
